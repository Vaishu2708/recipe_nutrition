# Recipe Nutrition Analyzer

A Django-based web application that calculates nutritional values of recipes
based on ingredients and quantities.

## Features
- User login & admin panel
- Add ingredients with nutrition values
- Create recipes
- Automatic calorie and nutrient calculation
- Django Admin dashboard

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

## How to Run
```bash
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
